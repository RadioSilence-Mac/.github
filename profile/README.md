## 🔍 Overview

<a href="#radioSilence">RadioSilence</a> is a simple, focused outbound firewall and network monitoring tool for Mac that gives users visibility into and control over which applications are making network connections — addressing the privacy concern that many applications communicate over the network in ways users never see or explicitly authorize.

The <a href="#radioSilence">per-application outbound firewall</a> blocks network connections on an application-by-application basis — selecting specific applications to block from making any outbound network connections while leaving other applications free to connect normally. This granular control addresses the common scenario where a particular application's network behavior is unwanted or unnecessary — a utility that doesn't need internet access, an application with excessive analytics or telemetry traffic, or any software whose network communication the user wants to prevent without uninstalling the application entirely. The <a href="#radioSilence">real-time network activity display</a> shows which applications are currently making network connections as they happen — providing visibility into background network activity that would otherwise be invisible, revealing which applications are phoning home, checking for updates, sending analytics, or otherwise communicating over the network without explicit user action.

The <a href="#radioSilence">simple one-click control</a> makes blocking an application from network access as easy as a single toggle — no complex firewall rule configuration, no understanding of network protocols or ports required, just selecting an application and blocking its network access directly. This simplicity distinguishes RadioSilence from more complex firewall tools that require technical networking knowledge to configure effectively. The <a href="#radioSilence">connection blocking enforcement</a> actually prevents the blocked application's network traffic at the system level — not just hiding or warning about connections, but actively preventing them from completing, providing genuine network access control rather than passive monitoring alone.</p>

<a href="#radioSilence">Privacy protection focus</a> addresses the specific concern of applications collecting and transmitting data without clear user awareness — giving users the ability to prevent this transmission regardless of what the application's privacy policy states, providing a technical enforcement layer beyond policy-based privacy controls. <a href="#radioSilence">Lightweight system integration</a> operates with minimal performance impact — the firewall and monitoring functionality runs efficiently without the system slowdown that more complex security software sometimes introduces.

<a href="#radioSilence">Persistent blocking rules</a> maintain blocked application configurations across system restarts — once an application is blocked, it remains blocked until explicitly unblocked, without requiring reconfiguration after every reboot. <a href="#radioSilence">Menu bar accessibility</a> provides quick access to RadioSilence's controls and current blocking status from the menu bar — checking and adjusting network access controls without opening a full application window.</p>
